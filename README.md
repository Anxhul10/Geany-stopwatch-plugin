## REQUIREMENTS
- geany-devel
- gcc

## INSTALLATION
```
git clone https://github.com/Anxhul10/Geany-stopwatch-plugin.git
cd Geany-stopwatch-plugin
./install.sh
```
## HOW TO USE

- Open Geany
- Go to: Tools -> Plugin Manager
- Enable Stopwatch plugin

- Then go to: Tools -> Stopwatch
